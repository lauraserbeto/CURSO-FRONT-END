O que é bug e debug?

Bug

Um problema que ocorreu no código, muitas vezes por erro do programador, impede o funcionamento do software

Debug

Método de encontrar e resolver o bug, em JavaScript temos diversas estratégias para isso

Validação

Técnicas utilizadas para ter o mínimo possível de bugs no software



Strict

Modo de desenvolvimento que deixa o JS mais rigoroso na hora de programar

Deve ser declarado no topo do arquivo ou de funções
